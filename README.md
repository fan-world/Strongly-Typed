Strongly Typed — Универсальный HTML5 Шаблон
Strongly Typed — это чистый, адаптивный и типографически ориентированный шаблон для создания современных веб-сайтов. Он идеально подходит для блогов, портфолио или корпоративных страниц, где контент стоит на первом месте.
🚀 Особенности
Адаптивный дизайн: Сайт отлично выглядит на смартфонах, планшетах и десктопах.
Акцент на типографику: Использование выразительных шрифтов для улучшения читаемости.
Секция "Hero": Привлекающий внимание заголовок и призыв к действию (CTA).
Блоки контента: Готовые решения для вывода последних постов, новостей и цитат.
Контактная форма: Встроенная форма обратной связи и иконки социальных сетей в футере.
🛠 Технологии
HTML5
CSS3 (Sass/SCSS)
JavaScript (jQuery)
Font Awesome (для иконок)
📁 Структура проекта
index.html — главная страница.
assets/css/ — файлы стилей.
assets/js/ — скрипты для интерактивности.
images/ — папка с изображениями и заглушками.
🎨 Кастомизация
Чтобы изменить основные цвета или шрифты, отредактируйте переменные в файле стилей assets/css/main.css. Шаблон построен по модульному принципу, что позволяет легко добавлять или удалять секции.
📞 Контакты
Секция "Get in touch" в нижней части страницы позволяет пользователям быстро связаться с вами. Не забудьте настроить обработчик формы в footer.
